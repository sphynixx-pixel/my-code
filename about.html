About Neon Drift

Neon Drift is a fantasy-inspired showcase bridging anime aesthetics and JDM culture. This project was built for a final assessment to demonstrate:

  • Semantic HTML structure
  • Responsive CSS design (Flexbox & Grid)
  • Dynamic rendering using JavaScript & DOM manipulation
  • Form validation and user interaction
  • Data persistence using localStorage
